# L11 Beginner Checkpoint

Before proceeding, confirm that you can:

- distinguish **Role**, **Person**, **Assignment**, **Capacity** and **Accountability**;
- identify the Functional Manager—not the PM—as approver of departmental capacity;
- produce a RACI with one accountable owner per decision;
- escalate only a capacity exception or priority conflict to Steering Committee.

If not, rebuild one row of the template before opening the assessment.
