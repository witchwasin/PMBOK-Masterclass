# Lesson 06 — Beginner Checkpoint

1. Stakeholder ต่างจาก Resource อย่างไร
2. Power/Interest เป็นข้อเท็จจริงถาวรหรือ assessment ตามบริบท
3. Current Engagement ต่างจาก Desired Engagement อย่างไร
4. PM ต้องทำให้ทุกคนพอใจหรือไม่
5. Resistance แบบใดควรแก้ด้วย process/capacity มากกว่า communication

## Routing

- ข้อ 1–2: ทบทวน Identify/Analyse Stakeholders
- ข้อ 3–5: ทบทวน Engagement Strategy และ PM Decision Thinking

ผ่านเมื่ออธิบาย 4/5 ข้อพร้อม evidence หรือ action ที่ตรวจได้
