# Model Answer — Lesson 03 Workshop

คำตอบระดับ Professional ต้องแยก ERP delivery phases ออกจาก 5 Process Groups และแสดงว่า Planning, Executing และ Monitoring & Controlling เกิดซ้ำใน Blueprint, Build/Migration และ Test

Data cleansing rule ที่ยังไม่ยืนยันเป็น planning/evidence gap การเริ่ม prototype เป็น executing activity และผล prototype ต้องถูก monitor เพื่อใช้ replan ไม่ใช่เหตุผลให้ข้าม approval gate

Closing ต้องรวม acceptance, handover, unresolved-item ownership และ benefits review ไม่ใช่เพียงจัด lessons learned

Lifecycle แบบอื่นยอมรับได้เมื่อใช้ boundary จาก Lesson 02 มี feedback loop และระบุ gate evidence กับ authority ชัดเจน
