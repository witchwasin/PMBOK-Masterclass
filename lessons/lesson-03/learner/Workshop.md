# Lesson 03 Workshop — Map the Management Flow

## Scenario

ERP อยู่ช่วง planning แต่ทีม SI เริ่ม prototype แล้ว Data Owner ยังไม่ยืนยัน cleansing rules และ UAT capacity ของ key users ยังไม่ชัด

## Task

1. ร่าง lifecycle phases ตั้งแต่ initiation ถึง benefits transition
2. วาง management activities ลงใน Process Groups
3. แสดงว่า Executing กับ Monitoring & Controlling เกิดคู่กันตรงไหน
4. ระบุ signal ที่ต้องกลับไป Replanning
5. กำหนด exit evidence และ approval authority ของแต่ละ gate

## Constraint

ห้ามใช้ Process Groups เป็นชื่อ phase ทั้งหมด และห้ามใส่ Planning เพียงครั้งเดียว

## Expected Output

Lifecycle and Process Group Map ที่ใช้เป็น Input ของ Cross-impact Map ใน Lesson 04
