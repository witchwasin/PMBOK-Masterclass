# Lesson 09 — Beginner Checkpoint

1. Estimate ต่างจาก Budget อย่างไร
2. Working Budget ต่างจาก Total Funding Envelope อย่างไร
3. Cost Baseline ต่างจาก Management Reserve อย่างไร
4. Actual Cost ต่างจาก Forecast/EAC อย่างไร
5. CPI/SPI ต่ำกว่า 1 บอกอะไรและยังไม่บอกอะไร

## Routing

- ข้อ 1–4: ทบทวน Estimate/Budget/Baseline/Reserve
- ข้อ 5: ทบทวน Earned Value Interpretation

ผ่านเมื่ออธิบาย 4/5 ข้อและไม่ปน Funding Envelope กับ Baseline
