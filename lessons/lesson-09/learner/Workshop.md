# Lesson 09 Workshop — Cost Performance Decision

## Scenario

ERP มี Working Budget 45 ล้านบาทและ Total Funding Envelope 60 ล้านบาท รายงานเดือนปัจจุบันแสดง CPI/SPI ต่ำกว่า 1 และมี Change Request 2 ล้านบาท

## Task

1. แยก funding, baseline, reserve, actual และ forecast
2. คำนวณ/ตีความ CPI, SPI, EAC และ VAC จากข้อมูลที่ให้โดย Instructor
3. ระบุสิ่งที่ตัวเลขยังไม่บอก
4. เสนอ approve, defer หรือ de-scope พร้อมผลต่อ value/schedule/quality
5. ระบุ authority และ baseline/forecast updates

## Constraint

ห้ามสรุปสถานะจาก % เงินใช้เทียบ % เวลาผ่านไปโดยไม่มี Earned Value และ estimate basis
