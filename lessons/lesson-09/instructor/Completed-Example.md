# Completed Example — ERP Cost Control Extract

## Funding Structure

- Working Budget: 45 ล้านบาทสำหรับ delivery/implementation
- Total Funding Envelope: 60 ล้านบาทรวม license และ management reserve ตาม Scenario Master
- ห้ามรายงาน 60 ล้านบาทเป็น Cost Baseline โดยไม่แสดงองค์ประกอบ

## Worked EVM Example

สมมติ BAC 45M, PV 18M, EV 15M, AC 20M:

- CPI = 15/20 = 0.75
- SPI = 15/18 = 0.83
- EAC แบบ cost-performance continuation = 45/0.75 = 60M

ตัวเลขเป็น teaching calculation ไม่ใช่ actual Scenario fact Recommendation ต้องถาม cause, remaining work, one-time variance, quality/data readiness และ reserve authority ก่อนตัดสิน change
