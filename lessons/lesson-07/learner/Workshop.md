# Lesson 07 Workshop — From Objective to WBS

## Scenario

CFO ขอ “รายงานบริหารทั้งหมด” ระหว่าง ERP Blueprint แต่ไม่มี user decision, report inventory หรือ acceptance criteria

## Task

1. แปลง need เป็นคำถามและ testable requirements
2. กำหนด Product Scope/Project Scope และ In/Out Scope
3. สร้าง WBS ระดับ 1–3 สำหรับ Reporting/Data workstream
4. เขียน WBS Dictionary อย่างน้อย 3 work packages
5. ระบุ approval/change route

## Constraint

ห้ามแตก WBS ตาม department หรือใส่ activity เช่น “ประชุม” เป็น deliverable
