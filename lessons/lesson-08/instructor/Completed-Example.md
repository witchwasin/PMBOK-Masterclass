# Completed Example — ERP Test/Migration Network

```text
SIT execution
→ critical defect fix
→ SIT exit evidence
→ migration rehearsal
→ reconciliation approval
→ UAT execution
→ UAT acceptance
→ cutover readiness
```

SIT delay 2 สัปดาห์ยังไม่พิสูจน์ว่า go-live delay 2 สัปดาห์ ต้องตรวจ float, parallel work, retest volume, key-user capacity และ migration dependency

Recovery options:

- Crash defect/retest ด้วย specialist เพิ่ม cost
- Fast-track migration preparation เพิ่ม rework risk
- Defer non-critical scope รักษา quality/date แต่ลด initial value

Steering Committee อนุมัติเมื่อ recovery กระทบ baseline/quality risk
