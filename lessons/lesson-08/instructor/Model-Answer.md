# Model Answer — Lesson 08 Workshop

Professional answer ต้องสร้าง logic chain ก่อนสรุปผลต่อ go-live ระบุสิ่งที่ยังไม่รู้ เช่น defect severity/retest effort, migration rehearsal dependency, UAT capacity และ current float

ตัวเลือก recovery ต้องแสดง schedule benefit พร้อม cost/quality/risk ไม่ใช้คำว่า “เร่งทีม” แบบไม่มี mechanism หากกระทบ baseline ให้ PM เสนอ recommendation และ Steering Committee อนุมัติพร้อม updated schedule evidence
