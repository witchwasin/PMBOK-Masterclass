# Lesson 08 — Beginner Checkpoint

1. Work Package ต่างจาก Activity อย่างไร
2. Milestone ต่างจาก Activity อย่างไร
3. Dependency ต่างจากวันที่ที่ผู้บริหารต้องการอย่างไร
4. Critical Path กับงานสำคัญที่สุดเหมือนกันหรือไม่
5. Float ใช้ตัดสินผลของ delay อย่างไร

## Routing

- ข้อ 1–3: ทบทวน Define/Sequence Activities
- ข้อ 4–5: ทบทวน CPM และ Schedule Control

ผ่านเมื่ออธิบาย 4/5 ข้อและวาด network ตัวอย่างได้
