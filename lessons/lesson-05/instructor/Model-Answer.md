# Model Answer — Lesson 05 Workshop

คำตอบระดับ Professional ต้องไม่อนุมัติหรือปฏิเสธจากราคาและเวลาเพียงอย่างเดียว ต้องขอ regulatory urgency, acceptance criteria, SI estimate basis, data/migration impact, UAT capacity และ benefit consequence

Change มีผลต่อ Scope, Schedule, Cost, Quality, Resource, Risk, Procurement และ Stakeholder จึงต้องผ่าน Steering Committee ตาม threshold ไม่ใช่ PM ตัดสินใจลำพัง

Recommendation ที่ยอมรับได้มีหลายแบบ แต่ต้องแสดง options และหลังอนุมัติต้อง update baselines, decision log, stakeholder communication และ next actions

หากระหว่างวิเคราะห์พบปัญหาที่เกิดแล้ว ให้เปิด Issue Log ในระบบบริหารงาน แต่การสอนรายละเอียดดังกล่าวอยู่ใน Lesson 13
