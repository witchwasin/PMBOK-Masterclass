# Lesson 05 — Beginner Checkpoint

1. Project Charter ต่างจาก Project Management Plan อย่างไร
2. Sponsor อนุมัติ Charter แล้ว PM มีอำนาจไม่จำกัดหรือไม่
3. Change Request ต่างจาก Issue Log อย่างไร
4. เหตุใด Change Decision ต้องวิเคราะห์หลาย baseline
5. หลังอนุมัติ Change แล้วต้องทำอะไรต่อ

## Routing

- ข้อ 1–2 ไม่ผ่าน: อ่าน Charter และ Governance
- ข้อ 3 ไม่ผ่าน: อ่าน Change/Issue Boundary
- ข้อ 4–5 ไม่ผ่าน: อ่าน Integrated Change Control และ Artifact Handoff

ผ่านเมื่ออธิบาย 4/5 ข้อและระบุ Approval Authority/Evidence ได้
