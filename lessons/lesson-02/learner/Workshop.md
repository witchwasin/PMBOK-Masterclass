# Lesson 02 Workshop — Project, Operation or Both?

## Scenario

CFO ต้องการลดเวลาปิดงบจาก 15 วันเหลือ 5 วัน มีสองข้อเสนอ: แก้รายงานรายเดือนเป็น BAU หรือทำ ERP Transformation 5 modules พร้อมปรับ process/data ownership

## Task

สร้าง Project vs Operation Analysis และ Value Chain หนึ่งหน้า โดย:

1. แยก monthly close, temporary report fix, ERP implementation และ post-go-live support
2. ระบุข้อมูลที่ยังขาดก่อนเลือก governance
3. แยก Output, Outcome, Benefit และ Value
4. ระบุ Sponsor, Benefit Owner และ Operation Owner
5. เสนอว่าควรตั้งเป็น project, operation improvement หรือทั้งสองส่วน

## Constraint

ห้ามสรุปจากการมี deadline หรือขนาดงบเพียงอย่างเดียว

## Expected Evidence

Approved classification, measurable value statement และ transition ownership
