# Scoring Rubric — Project vs Operation and Value Chain

| มิติ | Incomplete | Usable | Professional |
|---|---|---|---|
| Classification | ใช้ deadline/size ตัดสิน | ใช้ temporary/unique/ongoing | แยก transitional work และ governance consequence |
| Value Chain | หยุดที่ Output | ครบถึง Benefit/Value | มี baseline, evidence และ review date |
| Ownership | ไม่มี Owner | มี Sponsor/Operation Owner | แยก creator, benefit owner, approver ชัด |
| Evidence | เดา | ระบุ gap | จัดลำดับ evidence และ decision deadline |
| Recommendation | ไม่มี trade-off | เลือกพร้อมเหตุผล | มี options, risk และ next action |

ผ่านเมื่อทุกมิติอย่างน้อย `Usable`
