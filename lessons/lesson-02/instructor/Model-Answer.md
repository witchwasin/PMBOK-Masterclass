# Model Answer — Lesson 02 Workshop

คำตอบที่ผ่านต้องระบุว่า monthly close เป็น Operation แม้มี deadline ส่วน ERP Transformation เป็น Project เพราะเป็น temporary initiative ที่สร้างการเปลี่ยนแปลงเฉพาะตัว การแก้ report อาจเป็น operational improvement และอยู่คู่กับ ERP ได้หากมี Owner และขอบเขตแยกกัน

Value Chain ที่ถูกต้องไม่หยุดที่ “ระบบ Go-live” แต่เชื่อมไปยัง adoption, data quality และเป้าหมาย 15 → 5 วัน โดย CFO/Finance Process Owner เป็น Benefit Owner หลัง Handover

ข้อมูลที่ควรถาม ได้แก่ root cause, cost of delay, current adoption/data baseline, capacity key users, transition owner และเวลาที่ต้องเห็น benefit

ยอมรับ recommendation อื่นเมื่อผู้เรียนแสดง evidence, trade-off และไม่อ้างว่า solution สร้าง value อัตโนมัติ
