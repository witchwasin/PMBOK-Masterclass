# Review Checklist — Project vs Operation and Value Chain

- [ ] Classification ไม่อิง deadline อย่างเดียว
- [ ] Temporary, unique และ ongoing work แยกได้
- [ ] มี transition point และ Operation Owner
- [ ] Value Chain ครบ Need → Output → Outcome → Benefit → Value
- [ ] ทุก stage มี measure/evidence และ Owner
- [ ] ระบุ missing information ก่อน recommendation
- [ ] ตัวเลขตรง Scenario Master
- [ ] Recommendation ระบุ governance และ next decision
