# Model Answer — Lesson 04 Workshop

คำตอบระดับ Professional ต้องแสดง dependency เฉพาะเจาะจงและไม่เติมผลกระทบให้ครบช่องโดยไร้หลักฐาน อย่างน้อยต้องเห็น Scope, Schedule, Cost, Quality, Resource, Stakeholder, Risk, Procurement, Communication และ Integration decision

สามทางเลือกที่ยอมรับได้คือ include พร้อมเพิ่มเวลา/งบ, defer, หรือ replace lower-value scope แต่ recommendation ต้องผูกกับ High Season, outcome, integration readiness และข้อมูลที่ยังขาด

Product Owner/functional leads เป็น Impact Owners ในแต่ละด้าน ส่วน Sponsor/Change Authority เป็นผู้อนุมัติตาม threshold PM เป็นเจ้าของ integrated analysis ไม่ได้อนุมัติทุกเรื่องเอง
