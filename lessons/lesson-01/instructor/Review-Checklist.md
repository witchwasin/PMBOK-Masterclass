# Review Checklist — PM Learning Baseline

- [ ] ระบุ Business Need ก่อน solution
- [ ] แยก Output, Outcome, Benefit และ Value
- [ ] ระบุ Owner หลัง Handover
- [ ] ระบุ blind spot อย่างน้อย 2 เรื่อง
- [ ] ระบุ evidence gap และวิธีตรวจสอบ
- [ ] ไม่อ้างว่า PMBOK เป็นสูตรตายตัว
- [ ] ไม่ประกาศ success จาก Go-live อย่างเดียว
- [ ] ไม่มีข้อมูลขัด Scenario Master
