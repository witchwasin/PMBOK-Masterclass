# Scoring Rubric — Quality Artifact Pack

| มิติ | Incomplete | Usable | Professional |
|---|---|---|---|
| Traceability | test list ลอย | requirement→test | result→acceptance→handover ครบ |
| QA/QC | ปนกัน | แยก process/result | prevention/measurement/owner ชัด |
| Criteria | “ใช้งานได้” | measurable | threshold/evidence/authority ครบ |
| Test Strategy | defect count | levels/entry/exit/severity | data/environment/regression/readiness ครบ |
| Release Decision | สี dashboard | risk-based options | residual risk/rollback/approval evidence |

ผ่านเมื่อทุกมิติอย่างน้อย `Usable`
