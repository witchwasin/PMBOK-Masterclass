# Completed Example — ERP Finance Acceptance

| Requirement | Criterion | Evidence | Authority |
|---|---|---|---|
| Close status visibility | authorised Finance usersเห็นสถานะครบตาม approved fields | role test + UAT record | Finance Process Owner |
| Reconciliation accuracy | migrated balances reconcile within approved tolerance | signed reconciliation report | CFO delegate/Data Owner |
| Close-cycle readiness | critical close scenarios pass; open defects within accepted threshold | coverage + defect/risk record | CFO/Steering Committee |

Dashboard ต้องแสดง requirement coverage, severity/age, retest, data result, user readiness และ acceptance state สีเขียวอย่างเดียวไม่ใช่ evidence
