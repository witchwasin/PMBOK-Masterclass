# PMBOK Masterclass — Course Roadmap

## 1. Course Vision

หลักสูตรนี้ถูกออกแบบเพื่อให้ผู้เรียน **เข้าใจ Project Management อย่างเป็นระบบและนำไปใช้กับงานจริงได้** ไม่ใช่เพียงท่องจำคำศัพท์ Process หรือ Template

แนวทางการสอนของทุกบทคือ:

1. เริ่มจากคำถามว่า **ทำไมต้องรู้เรื่องนี้**
2. อธิบายปัญหาและที่มาของแนวคิด
3. สร้าง Mental Model ก่อนเข้าสู่ศัพท์และ Framework
4. ใช้กรณีศึกษา ERP Transformation และ Hotel Booking Digital Platform
5. เปรียบเทียบมุมมอง Enterprise Project กับ Digital Product
6. แก้ความเข้าใจผิดที่พบบ่อย
7. เชื่อม Output ไปสู่ Outcome, Benefit และ Business Value
8. ปิดด้วย Reflection และ Assessment

ทุกบทต้องสร้าง Artifact ต่อเนื่องด้วยวงจร `Learn → Watch PM Think → Watch Completed Artifact → Do → Review → Approve → Handoff` ตาม [Execution Baseline](EXECUTION-BASELINE.md) และ [Artifact Dependency Map](ARTIFACT_DEPENDENCY_MAP.md)

# 2. Course Structure Overview

หลักสูตรแบ่งเป็น 4 ช่วงหลัก

## Part I — Foundation

สร้างความเข้าใจว่า Project Management คืออะไร ทำไม PM ต้องรู้ PMBOK และโครงการไหลผ่านกระบวนการบริหารอย่างไร

- Lesson 01–04

## Part II — Core Project Management Knowledge

เจาะลึกองค์ความรู้หลักที่ Project Manager ต้องบริหารให้ครบ

- Lesson 05–14

## Part III — Adaptive Delivery

เข้าใจ Agile, Scrum, Kanban และความแตกต่างระหว่าง Predictive, Agile และ Hybrid

- Lesson 15–16

## Part IV — Applied Mastery

การประยุกต์ใช้ทั้งหมดจะถูกสอดแทรกอยู่ในทุกบทผ่านกรณี ERP และ Hotel Booking แทนการแยกเป็นบททฤษฎีล้วน
