# E-book Terminology Register

## Authority

Terminology must align with `governance/PM_GLOSSARY.md` and `governance/CONTENT-RULES.md`.

## Mandatory Distinctions

| Term pair or group | Required distinction |
|---|---|
| Project / Operation | Project is temporary and unique; Operation is recurring work |
| Output / Outcome / Benefit / Value | Output is what the project creates; Outcome is changed business state; Benefit is measurable gain; Value is realized business usefulness |
| Requirement / Scope | Requirement is a testable need; Scope defines included and excluded work |
| Deliverable / Work Package / Activity | Deliverable is a verifiable output; Work Package is lowest WBS unit; Activity is scheduled work with duration/resource |
| Estimate / Budget / Baseline / Forecast | Estimate is expected cost/time; Budget is approved funding; Baseline is approved control reference; Forecast is updated projection |
| Quality / Grade | Quality is fitness to agreed requirements; Grade is category/class of capability |
| QA / QC / Testing / Acceptance | QA prevents process defects; QC inspects outputs; Testing verifies behavior; Acceptance confirms agreed criteria are met |
| Risk / Trigger / Issue / Escalation / Change Request | Risk has not occurred; Trigger is a warning sign; Issue has occurred; Escalation asks for higher authority; Change Request seeks approved baseline change |
| Stakeholder / Resource | Stakeholder can affect or be affected; Resource is a person/team/material capacity assigned to work |
| Product Backlog / WBS | Product Backlog is prioritized product work; WBS decomposes project scope into deliverable-oriented work |
| Sprint / Phase | Sprint is a short iterative delivery cycle; Phase is a project segment often tied to governance/deliverables |
| Increment / Activity Output | Increment is usable product growth; Activity Output is the result of a scheduled task |
| Predictive / Agile / Hybrid / Tailoring | Predictive plans upfront; Agile iterates; Hybrid combines deliberately; Tailoring adapts governance to context |
